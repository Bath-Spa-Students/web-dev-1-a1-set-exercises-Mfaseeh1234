HTML Tags

<div>

The <div> tag is a generic container used to group elements together.

<div>
    <p>This is a paragraph inside a div.</p>
</div>

<h1> to <h6>

Heading tags are used to define headings. <h1> being the highest level and <h6> being the lowest.

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p>

The <p> tag represents a paragraph.

<p>This is a paragraph.</p>

<a>

The <a> tag creates hyperlinks to other web pages or resources.

<a href="https://www.example.com">Example Link</a>

<img>

The <img> tag embeds images into the document.

<img src="image.jpg" alt="Description of image">

<ul>

The <ul> tag creates an unordered list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>

The <ol> tag creates an ordered list.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<li>

The <li> tag represents a list item in ordered or unordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<span>

The <span> tag is used to group inline-elements in a document.

<p>This is a <span style="color: red">red</span> word.</p>

<table>

The <table> tag creates a table.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

CSS Properties

color

Sets the color of text.

body {
    color: #333;
}

font-size

Sets the size of the font.

p {
    font-size: 16px;
}

margin

Sets the margin of an element.

div {
    margin: 10px;
}

background-color

Sets the background color of an element.

header {
    background-color: #333;
}

text-align

Sets the alignment of text.

header {
    text-align: center;
}

list-style-type

Sets the style of list marker.

nav ul {
    list-style-type: none;
}

border

Sets the border properties of an element.

th, td {
    border: 1px solid #ddd;
}

padding

Sets the padding of an element.

th, td {
    padding: 8px;
}
